# Alcance MVP - City 9Twin

## Must have
- Login con roles.
- Dashboard con KPIs de movilidad.
- Zona piloto Chía - Unisabana - Cajicá.
- Fuentes de datos con estado de confianza.
- Creación y simulación de escenarios.
- Comparador básico de escenarios.
- Registro de incidentes.
- Auditoría de acciones críticas.
- Exportación básica CSV/PDF desde navegador.

- Mapa SVG con congestión por segmento.
- Indicadores visuales de confianza.
- Checklist de QA para navegación.

## Could have
- Integración real con APIs externas.
- Backend Spring Boot.
- Base de datos PostgreSQL/PostGIS.
- Docker Compose completo.

## Won't have en este corte
- App móvil.
- Machine learning real.
- Integración gubernamental real.
- Datos GTFS-RT en producción.
- Motor SUMO real conectado.
